# 🌸 Iris Flower Prediction App

This is a Machine Learning web app built using Streamlit.

## 🚀 Features
- Predict Iris species
- Simple UI
- ML model (trained using sklearn)

## 🧠 Input Features
- Sepal Length
- Sepal Width
- Petal Length
- Petal Width

## ▶️ Run Locally
```bash
pip install -r requirements.txt
streamlit run app.py
